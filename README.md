# Liskov Substitution Principle (LSP)

## Опис
Принцип підстановки Лісков означає, що об’єкти підкласів повинні
замінювати об’єкти базового класу без порушення роботи програми.

## Приклад
У репозиторії показано приклад з птахами:
- Bird — базовий клас
- FlyingBird, Penguin — підкласи
- Усі підкласи коректно замінюють Bird
